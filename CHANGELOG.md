# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-28

### Added
- Initial release of ProcessLab
- Node-based computational graph for dynamic systems
- Core node types: Constant, Add, Multiply, Divide, State
- Model class for managing computational graphs
- Simulation engine with time stepping
- Integration methods: Euler, RK4
- Event detection system (ThresholdDetector, SteadyStateDetector)
- Recording system for data capture
- CSV recorder for data export
- Terminal recorder for console output
- Matplotlib recorder for visualization
- Support for custom integrators
- Support for custom event detectors
- Comprehensive examples and documentation

### Features
- Flexible node-based architecture
- Multiple numerical integration methods
- Event-driven simulation capabilities
- Extensible recorder system
- Type hints throughout the codebase
- Python 3.8+ support
